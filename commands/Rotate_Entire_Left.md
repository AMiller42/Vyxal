# `'` - Rotate stack left

- Arity: 0
- In: NA
- Out: NA

Rotates the entire stack left. If there aren't any values on the stack, take implicit input.

## Usage

```
1 2 3 ' ║ ⟨2|3|1⟩
3 1 2 ' ║ ⟨1|2|3⟩
'       ║ ⟨<input>⟩
```

## Justification

Rotating the entire stack is a core stack mechanism, hence its inclusion in Vyxal.
